# Pygame Connect Four with AI 🔴🟡

這是一個使用 Python 與 Pygame 開發的圖形化四子棋遊戲。專案透過完整的**物件導向程式設計**封裝遊戲邏輯與介面渲染，並實作了具備預判能力的 AI 對手。

## 🌟 核心功能與技術

* **物件導向架構：** 將遊戲本體抽象化為 `ConnectFour` 與 `ConnectFour_AI` 兩個核心類別，有效管理棋盤狀態、玩家輪替與圖形渲染。
* **雙模式切換：** 支援「單機雙人對戰」與「玩家 VS 電腦」兩種遊玩模式。
* **Rule-based AI 演算法：**
  * **主動獲勝：** AI 會優先掃描棋盤，若有立即獲勝的落子點，將自動採取行動。
  * **防禦阻擋：** 若玩家在下一步即將連線獲勝，AI 會精準計算並落子阻擋玩家的攻勢。
* **GUI 介面與事件迴圈：** 運用 `Pygame` 處理畫面更新、幀率控制以及滑鼠點擊座標的事件監聽。
* **動態勝負判定：** 採用優化的二維陣列掃描演算法，每次落子後即時檢查水平、垂直與雙向斜線的連線狀態。

## 🛠 執行環境與依賴套件

* **開發語言：** Python 3
* **核心套件：** `pygame`, `random`, `sys`
